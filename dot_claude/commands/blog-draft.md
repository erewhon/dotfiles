# Blog Post Draft Generator

Produce a complete blog post draft in this format:

```markdown

[Opening paragraph - hook the reader with why this matters]

[Set up the situation. What challenge or question are we addressing?]

## [Section 2: The Core Content]

[Main explanation, tutorial, or argument]

### [Subsection if needed]

[Details, examples, code samples]

```code
// Include working code examples where relevant
```

## [Section 3: Practical Application]

[How to actually use this information]

## [Section 4: Common Pitfalls / Things to Watch For]

[What to avoid, gotchas, edge cases]

## Conclusion

[Summary of key points and clear next steps for the reader]

---

**Related Reading:**
- [Link to related content 1]
- [Link to related content 2]

**Tags:** tag1, tag2, tag3
```

3. **Include Meta Information**
   - Suggested SEO title
   - Meta description (150-160 characters)

Keep the tone conversational and practical. Focus on actionable takeaways.
