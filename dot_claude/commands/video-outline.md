# Video Outline Generator

Create a structured outline for a YouTube video on the topic: $ARGUMENTS

## Instructions

1. **Analyze the Topic**
   - What is the core value for viewers?
   - What level of knowledge does the audience have?
   - What's the ideal video length?

2. **Generate Outline**

Produce a video outline in this format:

```
TITLE OPTIONS:
1. [Title 1]
2. [Title 2]
3. [Title 3]

TARGET LENGTH: [X minutes]

---

## HOOK (0:00 - 0:15)
[Attention-grabbing opening - problem, question, or surprising statement]

## INTRO (0:15 - 1:00)
[What they'll learn, why it matters, quick credibility if needed]

## SECTION 1: [Topic] (1:00 - X:00)
- Key point 1
- Key point 2
- Demo/Example idea

## SECTION 2: [Topic] (X:00 - Y:00)
- Key point 1
- Key point 2
- Demo/Example idea

## SECTION 3: [Topic] (Y:00 - Z:00)
- Key point 1
- Key point 2
- Demo/Example idea

## CONCLUSION (Z:00 - End)
- Summary of key takeaways
- Call to action (subscribe, comment question, next video)

---

THUMBNAIL IDEAS:
1. [Visual concept 1]
2. [Visual concept 2]

CHAPTER MARKERS:
0:00 - [Hook/Intro]
1:00 - [Section 1 name]
...

B-ROLL/VISUAL NOTES:
- [Scene: Visual idea]
- [Scene: Visual idea]
```

3. **Add Production Notes**
   - What demos or code examples are needed?
   - What screen recordings should be prepared?
   - Any graphics or animations to create?

Keep the outline actionable and ready for scripting.
